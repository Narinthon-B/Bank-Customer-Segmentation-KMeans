# จุดประสงค์
การทำ **Customer Segmentation** (การแบ่งกลุ่มลูกค้า) โดยใช้ **K-Means Clustering** เพื่อวิเคราะห์พฤติกรรมการเงิน เช่น จํานวนเงินในบัญชี รูปแบบการใช้จ่าย และความต่อเนื่องในการใช้งาน เพื่อให้ธุรกิจสามารถระบุตัวตน (Personas) ของลูกค้าแต่ละกลุ่มได้อย่างชัดเจน และนำไปสู่การวางกลยุทธ์การตลาดที่ตอบโจทย์เฉพาะบุคคลได้อย่างแม่นยำ
